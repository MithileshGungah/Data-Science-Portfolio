# Retail Intelligence: Sales Analytics & Customer Segmentation

## Overview

This project combines retail sales analysis and customer segmentation to uncover business insights from transactional data.

The project includes:

- Exploratory Data Analysis (EDA)
- Data Cleaning & Validation
- RFM (Recency, Frequency, Monetary) Analysis
- Customer Segmentation using K-Means Clustering
- Business Insight Generation

---

## Objectives

### Superstore Sales Analysis

- Analyze customer segments and purchasing behavior
- Identify top-performing products and categories
- Explore sales trends over time
- Evaluate regional and state-level performance

### Online Retail Customer Segmentation

- Clean and validate transactional data
- Engineer RFM features
- Detect customer behavior patterns
- Segment customers using K-Means clustering
- Generate actionable marketing recommendations

## Key Insights

### Sales Analysis

- Consumer customers generated the highest revenue.
- California, New York, and Texas were the strongest markets.
- Phones and Chairs were among the top-performing product categories.
- Sales showed consistent growth over time.

### Customer Segmentation

Using RFM analysis and K-Means clustering, customers were grouped into:

| Segment | Description |
|----------|-------------|
| Retain | High-value loyal customers |
| Reward | Frequent and engaged customers |
| Re-engage | Customers showing reduced activity |
| Nurture | Lower-value customers with growth potential |

Additional outlier segments were identified for specialized marketing strategies.
